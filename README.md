# Cybersecurity Tools

Collection of Python scripts for security automation and network analysis.

## Tools

### 1. Port Scanner (`port_scanner.py`)
Simple TCP port scanner for identifying open ports on target systems.
- **Usage**: `python port_scanner.py &lt;target_ip&gt;`
- **Purpose**: Network reconnaissance and security assessment
